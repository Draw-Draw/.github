## 프로젝트 소개

그려그려는 생성형 ai Stable Diffusion을 통해 간단한 스케치만으로도 풍부한 그림일기를 만들어내는 웹 서비스입니다. <br/>
친구와 함께 일기를 공유하고 칭찬 도장을 찍으면서 여러분들의 동심을 되찾아보세요!
